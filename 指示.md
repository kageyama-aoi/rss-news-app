# 役割

あなたは経験豊富なフルスタックエンジニアです。
Vercel上で動作するニュース収集アプリを段階的に実装してください。

# 目的

RSSニュースを収集し、AI要約・保存・検索できるWebアプリを作る。

# 開発環境

* Vercel
* Next.js (App Router)
* Node.js
* JavaScript
* ローカル開発は `vercel dev`

# 実装ポリシー

以下の順番で段階的に実装すること。

Step1 → Step4まで段階的にコードを作る。
各ステップは **単体で動作確認できる状態**にする。

---

# Step1

複数RSS取得アプリ

要件

・複数RSS取得
・ニュース一覧表示
・クリックで記事リンク

RSS例

* Yahoo News
  https://news.yahoo.co.jp/rss/topics/top-picks.xml

* NHK News
  https://www3.nhk.or.jp/rss/news/cat0.xml

構成

```
app/
  page.js
api/
  rss/route.js
lib/
  rss.js
```

機能

API

```
GET /api/rss
```

返却

```
[
 {title, link, source}
]
```

---

# Step2

AI要約追加

記事タイトルをAIで要約する。

AI

OpenAI API

要約文字数

50文字以内

API

```
POST /api/summarize
```

入力

```
title
```

出力

```
summary
```

UI

ニュース一覧に

```
タイトル
AI要約
```

表示。

---

# Step3

ニュース保存

Supabaseを使う。

テーブル

```
news
```

カラム

```
id
title
link
summary
source
created_at
```

API

```
POST /api/news/save
GET /api/news
```

保存ボタンをUIに追加。
